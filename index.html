Module awesome-viewport

Manage tags based on viewports.

Once connected to a screen via viewport.connect(screen), selecting a single tag will make that tag the new viewport. This tag will remember any other selected tags when the viewport changes, so that when it is re-selected, all previously selected tags are viewed.
